Adiciona ao grafo uma aresta bidirecional (sob a forma de duas arestas simétricas) entre os vértices com id source e id dest, com capacidade capacity e serviço service.

Complexidade Temporal: O(V), sendo V o número de vértices do grafo

Parâmetros
sourceid de um dos vértices da aresta a adicionar
destid do outro vértice da aresta a adicionar
capacitycapacidade da aresta a adicionar
serviceserviço da aresta a adicionar
Retorna
true se a operação foi bem-sucedida, false caso contrário (não existe no grafo nenhum vértice com id source ou com id dest)

◆ relax()

bool Graph::relax ( Vertex *  u,
Vertex *  v,
Edge *  e 
)
staticprivate

Relaxa a aresta e, desde o vértice u até ao vértice v, i. e., se o custo do vértice v for superior ao custo do vértice u somado com o custo da aresta e, atualiza o custo do vértice v para este segundo valor e atualiza o predecessor do vértice v para a aresta e.

Complexidade Temporal: O(1)

Parâmetros
uvértice de origem da aresta e
vvértice de destino da aresta e
earesta a relaxar
Retorna
true se o relaxamento da aresta resultou num menor custo para o vértice v, false caso contrário
